脚本功能介绍：
Cobot扫描结果相关文档，从输出到结束分为3个：①Cobot导出后的原始文档（Excel，通常命名为“xxx的全部问题数据.xlsx”）②经由原始文档间接整理的过程文档（Excel）③经由过程文档最终整理为最终文档（Word）
本脚本用于将Cobot导出后的原始文档（文档①）的通用列，一键生成中间的过程文档（文档②）。以下称为①、②

【Cobot_processExcel.exe】可直接使用，无需安装Python环境！
操作步骤：
1.使用前请确保①与该exe同路径
2.双击运行，由于exe系pyInstaller打包，有15秒延迟属正常
3.弹出提示框“”，输入①文件名（如：saehrosdfcosch的全部问题数据.xlsx），将自动生成②的通用部分
